# Fresh with Hono

PoC project of Hono adapter for Deno's Fresh.

## Available endpoints

- JSON API: [`/api/todo`](https://fresh-with-hono.deno.dev/api/todo)
- Frontend UI: [`/todo`](https://fresh-with-hono.deno.dev/todo)

## Development

Launch a dev server with:

```
deno task dev
```
